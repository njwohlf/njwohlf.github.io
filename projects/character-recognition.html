English Handwritten Character Recognition

This project explores the multi-class supervised classification problem of recognizing handwritten characters "A" - "Z" in the English alphabet. Using the dataset from Cohen, Afshar, Tapson and van Schaik, I trained and compared two different neural network architectures to classify uppercase and lowercase English letters.

  • Deep Feedforward Network: Implemented a fully connected neural network to establish a baseline for character recognition performance. This model uses multiple dense layers to classify the handwritten characters.
  • Deep Convolutional Network: Developed a CNN architecture that leverages spatial patterns in the image data, resulting in improved recognition accuracy compared to the feedforward approach.
  • Comparative Analysis: Conducted detailed performance analysis between both models, examining accuracy, confusion matrices, and learning curves to identify strengths and weaknesses of each approach.
  • Data Processing: Utilized UTF-8 encoding for character representation and the StringEncodings package for data preprocessing, with visualization of training and test label distributions.
